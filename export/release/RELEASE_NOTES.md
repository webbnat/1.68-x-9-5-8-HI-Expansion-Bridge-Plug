## Design Change Notices

The `export/drawings/part_specs/*.md` shop sheets contained mutually
contradictory or physically impossible dimensions. The release geometry
resolves them as DCN-1..DCN-6. Each is also flagged on the affected
drawing sheet.

- **DCN-1 — Helix drive re-architected as a 2-start lead screw.**
  The spec put 0.080-deep helical grooves in a Ø1.552-ID insert riding a
  Ø1.550 mandrel body: zero wall. Resolution: the mandrel is necked to
  Ø1.270 over Z 10.4–23.0 (so the fixed SHEX-002 insert never sits over
  Ø1.550 body at any stroke position) with 2-start helical drive ribs at
  Z 19–23 (round form 0.115 W × 0.080 H, peak Ø1.429, lead 4.000).
  SHEX-002 ID changed 1.552 → Ø1.282 so the mating grooves have wall to
  live in (groove root Ø1.442).
- **DCN-2 — SHEX-011A tail sleeve.** Spec had OD 1.350 < ID 1.352
  (impossible). Resolution: ID 1.352 over the Ø1.350 tail, OD 1.550.
- **DCN-3 — Mandrel finished length 52.500.** The spec's Z 51–54 Ø1.550
  stub cannot coexist with the SHEX-012 Ø0.750 bore. Resolution: body to
  the joint shoulder at Z=51.0 plus a Ø1.500 × 1.5 pin stub.
- **DCN-4 — Plug overall length 57.0 in** with SHEX-012 Rev D 6.000 neck.
- **DCN-5 — SHEX-013 radial stack rebuilt.** Spec sleeve OD 1.680 could
  not enter the Ø1.562 body bore, and 0.070-deep grooves cut through
  every wall in the spec stack. Resolution: mandrel Ø1.350 tail extended
  through the sub zone (step moves Z 3.0 → 5.0); body OD 1.688 / ID 1.476
  with 2 internal gland grooves (0.045 deep) straddling the ports; sleeve
  OD 1.470 / ID 1.354 sliding inside the body over the seals.
- **DCN-6 — Top joint re-sequenced** so thread, seal and shoulder can all
  mate: mandrel shoulder at Z=51.0; pin Ø1.500 × 1.5 (1-1/2-12 UNF-2A
  thread 51–52, plain seal nose 52–52.5, Parker 2-216 face groove in the
  nose end face); SHEX-012 female: thread at mouth 0–1.0, seal bore
  Ø1.530 over 1.0–1.5.
- **DCN-7 — Settability: fishing neck downsized (Rev E).** The top joint was
  reshaped so a setting sleeve can pass over the neck and land on the upper
  slip carrier face at Z=51 (Ø1.375 pin, Rev E neck Ø1.450 body, GS profile
  retained for retrieval). Enables the SHEX-AK-20 adapter kit.
- **DCN-8 — Net set stroke re-budgeted 11.0 → 10.0 in** (+0.25 overtravel =
  10.25 demand) to fit a Baker E-4 No. 20 (10.777 in stroke) and #20-class CT
  tools (~10.5–11 in). Release stud calibrated to 45 klbf ±5%.
- **DCN-9 — Iris drive interference fix + module re-index.**
- **DCN-10 — Stent material.** SHEX-008/009 stage-1/2 stents re-specified from
  17-4 PH H900 to a **ductile expandable alloy** (annealed 316L or Ni alloy):
  a pressure/balloon-expandable mesh must plastically hinge ~2x; H900 is too
  brittle and would crack. (Stage 1/2 actuator package, SHEX-MAN-003.)
- **DCN-11 — Stage-2 prime mover.** Changed from a Belleville/swage wedge to a
  **second inflatable bladder**. A rigid expander starting <= stent ID 1.562
  cannot grow to back the Ø5.624 deployed stent in the 0.146-in annulus
  (geometrically impossible); the Belleville stack is retained only as a
  hold-open / anti-recoil preload. (SHEX-MAN-003.)
- **DCN-12 — First-stroke initiation & sequencing.** Replaced the passive
  hydrostatic-referenced burst-disk trigger (p1<p2 + time orifice), which has
  no position control and fires during run-in (proven, `trigger_design.py` §1),
  with **commanded initiation** (CT applied differential + ball seat, or e-line
  EFI gate) + **completion-gated sequencing** (each stent's bottom-out pressure
  shifts an arming sleeve that enables the next stage), with the **EQ pilot
  interlocked to close first**. Adds sub **SHEX-017** (+ 017A arming sleeve,
  017B reference piston); re-tasks SHEX-013/014C/015E/016A; defines fail-safe
  states. (SHEX-EM-001 / SHEX-MAN-003 Rev B.)

## Modelling notes / known limitations

- The run-in assembly represents the iris segments, petals and slips as
  nested collapsed arcs inside the Ø1.688 envelope. The source documents
  describe "stent-like crimping" of these rigid parts, which is not
  geometrically possible; the collapsed positions shown are the kinematic
  intent (parts stacked at minimum radius), not an elastic deformation.
- Stage-1/-2 stent sleeves (SHEX-008/-009) are exported in the run-in
  (laser-cut, un-expanded) state — that is the as-manufactured geometry.
  In the SET assembly they are shown at their expanded diameters with the
  cell pattern re-mapped to the expanded radius.
- The SET assembly includes a 5.5-in 17# casing section (ID 4.892) as a
  reference body named `REF-casing`. Delete or hide it for FEA meshing.
- STEP bounding boxes around the helical features (SHEX-002, SHEX-011,
  stent sleeves) read slightly oversize in some CAD packages because the
  B-spline control hulls extend past the true surface. Mesh-verified true
  extents: SHEX-002 is exactly L 4.000, OD 1.555, bore Ø1.282.
- Round-trip integrity verified for all 15 part files: 1 solid per file,
  0 orphan surfaces after re-import.
